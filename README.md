## Лабораторная работа №5

Cистема для линейной организации, написанная с помощью нескольких полиморфных абстрактных типов данных – Динамический массив, Связанный список и Последовательность.

## Описание коммитов 
feat - добавление нового функционала
docs - изменение/добавление документации
fix - исправление ошибок
chore - вспомогательные изменения, не влияющие на код и документация
refactor - правки кода без исправления ошибок
